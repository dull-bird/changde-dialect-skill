# 汉寿口音差异补充（太子庙/县城片区）

> 基础词汇表《常德方言词语汇1000》标题叫"常德方言"，但据汉寿本地用户指出，编者水兵实际记的更偏**桃源话**口音，跟汉寿话（尤其县城"龙阳话"）在用词和读音上有实质差异，不能直接当成汉寿话的底子用。这份文件加上 `hanshou-ipa-vocab.md` 才是汉寿话本身的依据，来源标注见文末。

## 使用方式（优先级）

用户明确表明自己是汉寿人（尤其太子庙、龙阳镇一带）时，**汉寿话数据优先于 `glossary.md`**，不是反过来：
1. 具体某个字/词怎么读、汉寿话有没有专用说法——先查 `hanshou-ipa-vocab.md`（语保工程官方 1200 条 IPA 词表，最权威）。
2. 查不到的字词、或需要语法/整体语感——参考本文件的语音规则和词汇表。
3. 只有以上两处都没有覆盖到的内容，才退回去参考 `glossary.md`（水兵的桃源话词表），且引用时心里清楚它是桃源口音，不是汉寿本地的。

## 官方权威分类 + 汉寿县内 5 种口音分区

中国语言资源保护工程（教育部、国家语委官方项目）采录展示平台对汉寿的官方调查点页面（调查点编号 26J41，https://zhongguoyuyan.cn/point/26J41 ，调查点负责人周婷，湖南文理学院）确认：

- **官方方言分类**：汉寿话属于"西南官话-湖广片-湘北小片"（注意这个官方分类跟学界一些论文里说的"常澧片"表述不完全一致，是分类体系版本差异，两者说的是同一片区域）。
- **汉寿县内部实际分 5 种口音**（调查点简介原文）："全县方言大体分5种口音，以县城为中心及周围乡镇居民说'龙阳话'，以崔家桥为中心及周边乡镇居民说'南乡话'，以丰家铺为中心及周围乡镇居民说'梅溪话'，以阳南塘为中心及周围乡镇居民说'安乐话'，西湖农场居民从安化、新化迁入，仍讲其故乡话（安化/新化话，属湘语）。"
- **太子庙定位**：太子庙镇紧邻汉寿县城（龙阳镇），属于"以县城为中心及周围乡镇"这一片，所以你老家话准确说应该叫**"龙阳话"**，是本文件目前描述的语音规则和词汇的主要适用范围；"南乡话""梅溪话""安乐话"是同县其他片区的说法，可能跟龙阳话有进一步细微差异，本文件暂未单独收录。

## 汉寿本地歌谣、俗语、农谚——官方语保平台原始采录

同一个调查点页面下收录了真实汉寿方言发音人的原始录音及文字记录（口头文化类，发音人金志丙，2019年8月10日采录于汉寿），是本文件唯一一批**逐字来自真实汉寿人口述、有采录日期和发音人姓名可考**的一手语料，比任何转述都更可靠：

儿歌/歌谣：
- "夜火虫儿夜夜来，家家搭信来。"
- "鸡鸡儿小，蛋蛋儿多，三岁的伢伢儿会唱歌。"
- "月亮粑粑跟到我走，走到南京打巴篓。"
- "打锣锣儿，嫁女儿。"
- "黄蜡嘚鸟儿满天飞，无娘的女儿本吃亏。"
- "我是汉寿一老倌嘚，讲话一口土话嘚。"

俗语/农谚：
- "先吃完的不管，慢吃完的洗碗。"
- "吃不言，睏不语。"（睏=睡）
- "聪明灵醒，要人提醒。"（**灵醒**=聪明机灵，新确认的汉寿本地词）
- "老鸹嘚嫌猪黑，自己不觉得。"（老鸹=乌鸦）
- "清明要明，谷雨要雨。"
- "芒种忙打苞，夏至把禾飙。"
- "小满不满，插它屙卵。"
- "月亮长哒毛，田里施得豪。"
- "檐米鸡儿飞得低，有雨在今夕。"
- "乌云接日连夜雨。"

情歌（旧时汉寿民间对唱调，语言较文，方言色彩不如上面俗语浓，但可作为地方文化引用）：
- "一天不见哥哥的面，大路人马都问遍。"
- "天上落雨地下滑，情哥哥教我种芝麻。"
- "草鞋烂哒四根筋，卡马嘚死哒四脚抻。"

从这批语料能进一步确认的语法点：**"哒"＝了、"嘚"＝的**，跟基础词表 `glossary.md` 记录的完全一致，是真实汉寿人口语里实际在用的，不是转写误差。

## 语音规则（比单个词更重要，影响一整类字的读法/写法直觉）

- **n/l 不分**：来母、泥母混读，"你"和"李"这类字发音不分。方言转写时 n 开头和 l 开头的字可以互相借用，不必严格区分。
- **无翘舌音**：没有 zh/ch/sh 卷舌声母，全部读成平舌（z/c/s 或近似音）。
- **fu/hu 合流为 f**：例如"府"和"虎"同音，"唤"和"饭"同音。凡是普通话 hu 开头、跟合口呼相拼的字，汉寿话里往往读成 f 声母。
- **合口呼简化为开口呼**：不少 -uan/-u- 介音的字丢掉圆唇介音，比如"端"（tuan）读成"tan"，跟"单"同音。
- **古全浊声母清化，古全浊上声归入去声**：跟大多数西南官话一致，声调系统里没有单独的浊上声调类，一般是 5 个调类。
- **阴去与阳平合流**：这是汉寿区别于桃源话的关键点——桃源话阴去、阳去两类调分化独立，汉寿话阴去并入阳平，调类因此比桃源少。

| 普通话 | 汉寿话 | 备注 |
|---|---|---|
| 个（量词） | 折 | 泛用量词，替代"个" |
| 妈妈 | 恩妈 | 称谓，比"姆妈"更汉寿本地 |
| 外婆 | 嘎嘎 | 与常德市区通用词一致 |
| 哪儿 | 何里 | 与常德市区通用词一致 |
| 这儿 | 德里 | 注意不是"这巴基" |

## 特殊片区现象（了解即可，不一定适用太子庙）

汉寿县军山铺、白鹿桥一带（偏老派口音）古全浊声母读边音 l，出现"钱/连不分，寿/路不分，陈/龙不分，蛇/茶不分"的现象；桃江/汉寿/安化交界的三堂街话则更特殊，古全浊声母多读零声母。这些是汉寿境内更细分的片区特征，太子庙/龙阳镇县城片区一般不适用，仅供了解汉寿方言内部并不统一这一事实。

汉寿东部靠近沅江、益阳的部分乡镇实际使用湘语（长益片-益沅小片），跟本文件描述的西南官话常澧片是两种不同的方言，本文件不覆盖那部分区域。

## 汉寿话人称代词——1935/1974 大型方言调查原始数据

李蓝《湖南方言的人称代词》，《中国语文研究》2002年第2期（总第14期），第63-74页，全文可公开获取：https://www.cuhk.edu.hk/ics/clrc/scl_14/li.pdf 。这篇论文直接使用《湖南方言调查报告》（赵元任、杨时逢等，1935年调查、1974年整理出版，覆盖湖南75个方言点）的第一手记音数据，汉寿是这75个调查点之一，论文里**多次单独列出汉寿的读音**，是本文件目前拿到的最接近"原始专业调查数据"的内容：

- **"我"**：汉寿归入读音"ᵑo"（鼻化 o，近似"额哦"连读）这一大组，跟长沙、湘潭、益阳、安化、临澧、澧县、安乡、沅江、南县等同组，是湖南方言里最常见的"我"读音类型。
- **"你"**：汉寿归入"ⁿi/ˡi"类的一组（跟长沙、益阳、桃源、慈利等同组）。
- **"他"**：汉寿归入"tʰa"（近似"他"的官话读音）一组，跟长沙、湘潭、宁乡、益阳、桃源、慈利、临澧、澧县、安乡等同组——说明汉寿"他"的读音比较接近常德市区，没有特别偏离。
- **人称复数（们）——这是汉寿真正独特的地方**：论文专门用一张表（表13）把汉寿和华容单独拎出来对比，因为这两个点的人称复数构词方式在湖南方言里比较少见。汉寿的规律是：
  - "我们"用一个鼻音后缀（论文记音近似 nan/ⁿjɛn）加在"我"后面构成，且**这个后缀被直接借用去构成"你们""他们"**——也就是说汉寿话里"你们""他们"字面上更接近"你（我们尾巴）""他（我们尾巴）"，而不是普通话那样"你/他"分别独立加"们"。
  - 论文原话："澧县、安乡、华容、汉寿四处第二、三人称的复数词都是'我们'。'你我们''他我们'的情况说明一定是第一人称先产生复数，然后被第二、三人称移用。"
  - 这说明汉寿话的"你们/他们"不是简单的"你+们/他+们"，而是历史上先有"我们"这个词，后来被整个搬去当"你们""他们"的复数标记——跟澧县、安乡、华容同属一类，是常澧片内部一个真实、罕见、可考证的语法特征，桃源、常德市区都不属于这一类（论文里桃源单独归在另一类，常德市区归在"单复数不同音但不用后缀复数词"的类型）。

## 沧山话（汉寿县沧港镇）疑问代词——已发表学术论文摘要

徐多懿、徐前师，《常德沧山话疑问代词研究》，《湖南科技大学学报（社会科学版）》2021年第2期，第156-162页。沧港镇隶属汉寿县，是本文件目前能查到的**唯一一篇专门研究汉寿境内某个具体乡镇方言、公开可查证的学术论文**（全文在期刊官网有付费墙，以下是能检索到的摘要内容）：

- 沧山话属西南官话，跟普通话及其他方言相比，疑问代词的分布、用法有自己的特点。
- **"何"系疑问代词分布最多**——即"哪儿/什么/怎么"这类疑问词，沧山话里大量用"何"打头的形式（跟本文件前面记录的"何里"=哪儿是同一个系统），比普通话保留了更多古汉语疑问代词成分。
- **"么的"是常德方言、也是沧山话的特色词**——对应 `glossary.md` 里已有的"么得"（什么），可以确认这个词在汉寿沧港一带同样通行，不是常德市区独有。
- 沧山话疑问代词跟"常德市区话"有明显不同，反而跟湘语、赣语、吴语有不少共性，学界认为跟方言互相影响和历史移民路线有关——从侧面印证了前面提到的"汉寿方言内部不统一，部分片区偏湘语影响"这个事实。
- 论文全文（含具体疑问代词逐条对照表）未能通过公开检索获取，如果找到全文渠道值得回来补充。
