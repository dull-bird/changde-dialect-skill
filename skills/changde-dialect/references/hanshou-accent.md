# 汉寿口音差异补充（太子庙/县城片区）

> 基础词汇表《常德方言词语汇1000》整理自常德话大类，属于"常澧片"西南官话，跟汉寿话同一个方言大类，但汉寿（尤其太子庙、龙阳镇等县城周边片区）在语音规则和少量词汇上有自己的specifics。这份文件是**补充/修正层**，不是独立词表——网上没有查到成体系、成规模的"汉寿方言专业词典"，本文件只收录了能交叉验证到的可靠差异点，来源标注见文末。

## 使用方式

用户明确表明自己是汉寿人（尤其太子庙、龙阳镇一带）时，在套用 `glossary.md` 基础词汇的同时，优先应用下面的语音规则和词汇替换，让回应更贴近汉寿本地口音，而不是笼统的"常德话"。

## 语音规则（比单个词更重要，影响一整类字的读法/写法直觉）

- **n/l 不分**：来母、泥母混读，"你"和"李"这类字发音不分。方言转写时 n 开头和 l 开头的字可以互相借用，不必严格区分。
- **无翘舌音**：没有 zh/ch/sh 卷舌声母，全部读成平舌（z/c/s 或近似音）。
- **fu/hu 合流为 f**：例如"府"和"虎"同音，"唤"和"饭"同音。凡是普通话 hu 开头、跟合口呼相拼的字，汉寿话里往往读成 f 声母。
- **合口呼简化为开口呼**：不少 -uan/-u- 介音的字丢掉圆唇介音，比如"端"（tuan）读成"tan"，跟"单"同音。
- **古全浊声母清化，古全浊上声归入去声**：跟大多数西南官话一致，声调系统里没有单独的浊上声调类，一般是 5 个调类。
- **阴去与阳平合流**：这是汉寿区别于桃源话的关键点——桃源话阴去、阳去两类调分化独立，汉寿话阴去并入阳平，调类因此比桃源少。

## 确认的词汇/语法差异

| 普通话 | 汉寿话 | 备注 |
|---|---|---|
| 个（量词） | 折 | 泛用量词，替代"个" |
| 妈妈 | 恩妈 | 称谓，比"姆妈"更汉寿本地 |
| 外婆 | 嘎嘎 | 与常德市区通用词一致 |
| 哪儿 | 何里 | 与常德市区通用词一致 |
| 这儿 | 德里 | 注意不是"这巴基" |
| 那边 | 洛里 | 注意不是"那巴基" |
| 很/非常 | 梆硬、喷香、交湿、灰滚的 | 程度副词，按语境选用，比"几得"更地道 |
| （惊讶感叹） | 啊喝 | 用于突发意外/不好的事，语气词 |

## 特殊片区现象（了解即可，不一定适用太子庙）

汉寿县军山铺、白鹿桥一带（偏老派口音）古全浊声母读边音 l，出现"钱/连不分，寿/路不分，陈/龙不分，蛇/茶不分"的现象；桃江/汉寿/安化交界的三堂街话则更特殊，古全浊声母多读零声母。这些是汉寿境内更细分的片区特征，太子庙/龙阳镇县城片区一般不适用，仅供了解汉寿方言内部并不统一这一事实。

汉寿东部靠近沅江、益阳的部分乡镇实际使用湘语（长益片-益沅小片），跟本文件描述的西南官话常澧片是两种不同的方言，本文件不覆盖那部分区域。
