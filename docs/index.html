🗣️常德话 Skill

用常德话跟 Claude Code / Codex / OpenClaw / Kimi Code CLI 聊天的 Agent Skill

基于《常德方言词语汇1000》（水兵1986编）整理：994 条日常词条 + 151 条桃源本地歇后语，配四平台都验证跑通的关键词开关钩子——说「说常德话」就开，说「说普通话」就关。这份底本标题叫"常德方言"，实际口音偏桃源话；仓库另配了一套独立验证的汉寿话（龙阳话）官方数据，汉寿用户优先用那套。

工具钩子机制验证方式状态

Kimi Code CLI~/.kimi/config.toml 的 hooks = [{ event = "UserPromptSubmit", ... }]（同一份 Python 脚本）kimi --print 真实跑通，session 状态文件落盘确认✓ 已验证

安装

三种方式；只有前两种能装上钩子（确定性开关），第三种只装技能本身

① 让 AI agent 自动装 技能 + 钩子

把下面这段发给 Claude Code / Codex / OpenClaw / Kimi Code CLI，它会克隆仓库、按你装的工具装好技能和对应钩子：

请帮我安装 changde-dialect 这个常德话 Agent Skill：
1. 克隆 https://github.com/dull-bird/changde-dialect-skill
2. 运行 ./install.sh，把 skills/changde-dialect 装到本机检测到的
   Claude Code / Codex / OpenClaw / Kimi Code CLI 的 skills 目录
3. 如果我用 Claude Code，运行 claude-code/install-hook.sh 注册关键词开关钩子
4. 如果我用 Codex，运行 codex/install-hook.sh（首次可能要批准 hook trust；
   非交互场景可以加 --dangerously-bypass-hook-trust，但这是 DANGEROUS 选项，
   装之前跟我确认一下）
5. 如果我用 OpenClaw，运行 openclaw/install-hook.sh
6. 如果我用 Kimi Code CLI，运行 kimi/install-hook.sh
7. 装完后跟我说"说常德话"确认效果，测试完再说"说普通话"退出

② 手动 clone + install.sh 技能 + 钩子

git clone https://github.com/dull-bird/changde-dialect-skill.git
cd changde-dialect-skill
./install.sh                    # 装技能到检测到的所有工具
./claude-code/install-hook.sh   # 可选：Claude Code 关键词开关钩子
./codex/install-hook.sh         # 可选：Codex 关键词开关钩子
./openclaw/install-hook.sh      # 可选：OpenClaw 关键词开关钩子
./kimi/install-hook.sh          # 可选：Kimi Code CLI 关键词开关钩子

③ npx skills 一键装 仅技能，不含钩子

用 skills.sh 的通用安装器，实测能正确识别并装好 skills/changde-dialect，自动适配 Claude Code / Codex / Cursor / OpenClaw / Kimi Code CLI 等 70 余个 agent；但它只拷贝技能文件，不会帮你注册确定性开关钩子（想要钩子还是走①或②）。

npx skills add dull-bird/changde-dialect-skill

代码、命令、报错等技术性输出不会被方言化。骂人/粗语词条只用于轻度打趣或用户主动询问词义，不用来真正冒犯人。
